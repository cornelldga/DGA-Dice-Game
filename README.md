INSTRUCTIONS TO OPEN UNITY PROJECT:

1) Download Github Desktop
2) In Github, click "Code" (Green Button on main page of repository), click "Local", and "Open with Github Desktop"
3) Download the latest Unity Hub version from https://unity.com/download
4) In Unity Hub, go to "Installs" and click "Install Editor" to download the latest LTS (long-term support) release of Unity
5) In Unity Hub, go to "Projects", click "add" on the top right and click "Add Project From Disk", choosing the repository folder (Should be called DGA-Dice-Game)
6) Open project with Unity Hub
